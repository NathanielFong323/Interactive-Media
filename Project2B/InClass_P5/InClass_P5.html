Week 04

Work 4.01

We learned the basic to P5.js, such as applying a color to the background, adding color and border to shapes and draw them across the canvas with a mouse, while making them slowly disappear.



Work 4.02

Working upong the basic we learnt from Work 01, we saw the creative possibilities through random color and shapes generation.



Work 4.03

We learnt how to import image into P5.js and reference them while drawing with a mouse. We also experimented on drawing without deleting previous frames, resulting in interesting patterns.



Week 05

Work 5.01

Building on our Week 04 works, we combined multiple techniques to create an interactive color revealing canvas. By utilizing image imports and a new mouse click detection function, we developed a system where dragging the cursor erases portions of a monochrome overlay, exposing the original colored image beneath.



Work 5.02

Similar to Work 4.03, we created drawings by tracking mouse movements while deleting previous frames. This time, however, we also used live webcam input and applying a filters to generate a engaging visual effects in real time.



Work 5.03

To me, this work marked a significant leap in complexity. Not only did we import fonts, but we also learned to control circle density along letters while making them jitter and change colors randomly each frame. Additionally, we implemented lines that extends from each circle to the mouse position, creating interactive visuals. This project inspired me as it demonstrated how combining various techniques we had learned could result in interesting media.



Work 5.04

In this work, we learned how to use a looping function and variables to generate circles at random while preserving the original image's colors. This technique created a mesmerizing pattern with an artistic, almost filter-like effect. Overall, I discovered the creative possibilities of random circle generation and how it can produce stunning visual compositions.



Week 06

Work 6.01

In Week 06, the complexity of our work increased once again. This time, we learned how to record sound using a microphone and play MP3 files. From these audio sources, we extracted amplitude data and mapped their values to a range of 5-20. These values could then control the size and rotation of objects in our scene. At the end, I chose to use the MP3 playback for control, as I found it created a more humorous effect. However, I incorporated both microphone and MP3 options in my final experiments, as I believe real-time volume visualization can produce stunning results.



Work 6.02

For our final in class p5.js project, we explored button interactions.We created a system where each button click generates another button in an endless loop. While the concept seems simple, the logic behind is surprisingly complex. This exercise ultimately taught me programming logics that I can apply to my own interactive media projects.
